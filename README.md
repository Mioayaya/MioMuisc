# miomusic
react+ts+vite+electron 搭建音乐项目


# 代码规范

- 函数名 下划线命名法  a_function()
- 组件名 Mio+大驼峰  MioHome
- 文件名 小写 + '-'  aa-bb-cc
- 接口 MIxxx  MIstyleLoad
- 变量 小驼峰
- 常量 大写

## 文件夹
- src
  - assets 美术资源
  - components 通用组件
  - hooks hooks
  - pages 页面
  - router 路由
  - server api
    - axios axios
    - local-data 本地数据
  - store 全局变量
  - type 类型接口
    - interface
  - utils 工具函数
